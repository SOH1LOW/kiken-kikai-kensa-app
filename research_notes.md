# 技能検定機械検査3級 調査結果

## 試験概要
- 中央職業能力開発協会が実施する国家資格試験
- 学科試験と実技試験がある
- 学科試験は◯×問題を含む形式

## 問題データソース
- 中央職業能力開発協会の公式サイトに過去問題が公開されている
- 令和4年度〜令和7年度の問題が閲覧可能
- PDFファイルで提供されているが、著作権保護のため閲覧のみ可能

## アプリ開発方針
実際の過去問題は著作権で保護されているため、以下の方針でアプリを開発する:

1. **問題データの作成**: 実際の技能検定機械検査3級の出題範囲に基づいた、オリジナルの◯×問題を作成
2. **出題範囲**: 
   - 測定機器の使用方法
   - 硬さ試験の種類と方法
   - 寸法測定の基礎
   - 検査の基本原則
   - 測定誤差と精度
   - 各種測定器具の特性

3. **問題形式**: 
   - ◯×2択問題
   - 1回のテストで30問をランダム出題
   - 間違った問題には解説を表示

4. **サンプル問題例**:
   - 「ショア硬さ試験は、ダイヤモンド円すい圧子を用いて硬度を測る試験方法である。」→ × (ダイヤモンドのハンマーを試験片に対して直角に落とし、ハンマーが跳ね返る高さを測定する)
   - 「マイクロメータは、ノギスよりも高精度な測定が可能である。」→ ◯
   - 「ブリネル硬さ試験では、鋼球圧子を用いる。」→ ◯

## 技術スタック
- React Native + Expo
- ローカルストレージ (AsyncStorage) でテスト履歴を保存
- データベース不要（問題データはアプリ内に静的に保持）
